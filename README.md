# Automatic Feedback Responder for Wildberries

Это веб-приложение для автоматического получения отзывов и вопросов с платформы Wildberries и генерации ответов на них с помощью ИИ.

## Возможности

- Автоматическая загрузка новых (безответных) отзывов и вопросов.
- Генерация ответов с помощью нейронной сети.
- Возможность редактировать, перегенерировать или изменять сгенерированные ответы.
- Простой и понятный веб-интерфейс.

## Настройка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <repository_url>
    cd automatic-feedback-responder
    ```

2.  **Настройте API-ключи:**
    Создайте файл `.env` в директории `backend` и добавьте в него свои ключи:
    ```
    WB_API_KEY="ВАШ_API_КЛЮЧ_WILDERRIES"
    ```
    - `WB_API_KEY`: Ваш токен для доступа к API Wildberries. Убедитесь, что у него есть права на **"Отзывы и вопросы"**.

3.  **Установите зависимости и запустите проект:**
    
    Откройте терминал в корневой папке проекта.
    
    a. **Установите `uv` (если он не установлен):**
    ```bash
    pip install uv
    ```

    b. **Создайте виртуальное окружение и установите зависимости:**
    Перейдите в папку `backend`:
    ```bash
    cd backend
    ```
    Создайте виртуальное окружение с помощью `uv`:
    ```bash
    uv venv
    ```
    Активируйте окружение. На Windows:
    ```powershell
    .venv\Scripts\Activate.ps1
    ```
    На macOS/Linux:
    ```bash
    source .venv/bin/activate
    ```
    Установите зависимости из файла `requirements.txt`:
    ```bash
    uv pip install -r requirements.txt
    ```

    c. **Запустите веб-сервер:**
    Находясь в папке `backend`, выполните:
    ```bash
    uvicorn app.main:app --reload
    ```

4.  **Откройте приложение в браузере:**
    Перейдите по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
